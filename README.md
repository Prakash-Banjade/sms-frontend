# School Management System Frontend

## Technologies Used

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React Hook Form](https://react-hook-form.com/)
- [Zod](https://zod.dev/)
- [Shadcn UI](https://ui.shadcn.com/)
- [React Query](https://tanstack.com/query/latest/)
- [React Router DOM](https://reactrouter.com/)
- [React To Print](https://github.com/niklasvh/html2canvas-react)

## Features

1. User Management
2. Roles & Permissions
3. Student, Teacher, Staff Management
4. Finance Module (Salary, Payments, etc.)
5. Library Module (Book Management, Issue, Return, etc.)
6. Attendance Module (Student, Teacher, Staff)
